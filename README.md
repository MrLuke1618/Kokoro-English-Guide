# Magical English Guide

An interactive web application designed to help learners, especially those in the medical field, improve their English language skills through engaging mini-games and helpful widgets.

### Features
* **Interactive Learning Games:** Includes mini-games for improving vocabulary, grammar, reading, and speaking.
* **Integrated Music Player:** A custom music player to provide a relaxing learning environment.
* **Live Widgets:** Displays the current date, time, and weather based on the user's location.
* **User-Friendly Interface:** A clean, magical-themed design to make learning fun and accessible.

### Technologies Used
* HTML
* CSS
* JavaScript
* WeatherAPI.com

### How to Use
To run this project, simply download the files and open the `index.html` file in your web browser. Make sure you have the correct API key from WeatherAPI.com in the `js/weather-widget.js` file for the weather widget to function.
